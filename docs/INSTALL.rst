sequoia.theme Installation
--------------------------

To install sequoia.theme using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``sequoia.theme`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        sequoia.theme

* Re-run buildout, e.g. with:

    $ ./bin/buildout
